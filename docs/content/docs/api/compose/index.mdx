# Compose Module

The `compose-camera-ui` module provides ready-to-use UI components for integration with Jetpack Compose.

## Installation

```kotlin
implementation("io.github.l2hyunwoo:compose-camera-ui:1.2.2")
```

## Key Components

- **[CameraPreview](./compose/camera-preview)**: The primary composable for displaying the camera feed.
- **[rememberCameraController](./compose/remember-camera-controller)**: A helper to create and remember a `CameraController`.
- **[FocusIndicator](./compose/focus)**: Default and custom focus indicators.
