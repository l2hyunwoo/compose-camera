# Capture Use Cases

## Image Capture

The default `ImageCaptureUseCase` handles taking photos.

### `ImageCaptureConfig`

- `flashMode`: Override flash mode for this capture.
- `quality`: JPEG quality hint.

## Video Capture

The `VideoCaptureUseCase` handles recording video.

### `VideoCaptureConfig`

- `audioEnabled`: Whether to record audio.
