# `FlashMode`

Controls the flash behavior during image capture and preview.

## Values

- `ON`: Flash will always fire during capture.
- `OFF`: Flash will never fire.
- `AUTO`: Flash will fire automatically if the scene is too dark.
- `TORCH`: The flash unit is kept on continuously (like a flashlight) during preview and capture.
