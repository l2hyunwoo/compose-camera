# `PermissionDenied`

Thrown when a required permission (usually `CAMERA` or `MICROPHONE`) has been denied by the user or the system.

**Common Causes:**

- User rejected the permission dialog.
- Permissions were not properly declared in `AndroidManifest.xml` or `Info.plist`.
