# Core Module

The `compose-camera-core` module provides the fundamental building blocks for camera integration. It abstracts the platform-specific implementations (CameraX for Android, AVFoundation for iOS) into a unified, Kotlin-first API.

## Installation

```kotlin
implementation("io.github.l2hyunwoo:compose-camera-core:1.2.2")
```

## Key Components

- **[CameraController](./core/camera-controller)**: The main entry point for controlling the camera (taking pictures, recording video, etc.).
- **[CameraConfiguration](./core/camera-configuration)**: Data class defining the desired camera state (lens, flash mode, etc.).
- **[CameraControl](./core/camera-control)**: Interface for runtime camera adjustments (zoom, focus, exposure).
- **[CameraInfo](./core/camera-info)**: Provides read-only information about the active camera (capabilities, state).
