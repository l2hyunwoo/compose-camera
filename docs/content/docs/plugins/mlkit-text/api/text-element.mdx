# `TextElement`

Represents a single word or symbol.

```kotlin
data class TextElement(
    val text: String
)
```

## Properties

| Property | Type     | Description                            |
| :------- | :------- | :------------------------------------- |
| `text`   | `String` | The localized string for this element. |
